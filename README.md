## 🧠 Overview

**Taskify** is a sleek and modern **task management web app** built using **HTML, CSS, and JavaScript**.
It helps users stay productive by managing daily tasks with an intuitive UI, smooth animations, and powerful productivity features.

---

## ✨ Features

* ➕ Add tasks instantly
* ✅ Mark tasks as complete / incomplete
* ✏️ Edit tasks anytime
* 🗑️ Delete tasks with smooth animations
* 🔍 Filter tasks (All / Active / Completed)
* 💾 LocalStorage persistence (no data loss)
* 📅 Due date tracking with overdue highlights
* 📊 Progress tracking bar
* 🌓 Dark / Light mode toggle
* 🖱️ Drag & drop task sorting
* ✨ Smooth UI animations & transitions

## 🚀 Getting Started

### 📥 Clone the repository

```bash
git clone https://github.com/Elijahamet/taskify.git
cd taskify
```

---

### ▶️ Run the project

#### Option 1 (Simple)

Just open:

```bash
index.html
```

in your browser.

---

#### Option 2 (Dev server)

```bash
npm install
npm run dev
```
